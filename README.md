# Profit Analysis Dashboard

## Project Overview
This project is a Profit Analysis Dashboard developed using Microsoft Power BI and Microsoft Excel. The dashboard helps analyze sales performance, profitability, product performance, and monthly sales trends through interactive visualizations.

## Tools Used
- Microsoft Power BI
- Microsoft Excel
- DAX (Data Analysis Expressions)

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Cost KPI
- Total Orders KPI
- Profit Margin Percentage
- Profit by Category (Bar Chart)
- Monthly Sales Trend (Line Chart)
- Sales by Region (Pie Chart)
- Sales by Product (Column Chart)

## Skills Demonstrated
- Data Cleaning
- Data Transformation
- Data Visualization
- DAX Measures
- Dashboard Design
- Business Intelligence

## Dataset
A sample stationery sales dataset was created in Microsoft Excel and imported into Power BI for analysis.

## Project Outcome
The dashboard provides a simple and interactive way to monitor sales performance, identify profitable product categories, and understand monthly sales trends to support better business decisions.
